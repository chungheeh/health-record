# W.E (Workout & Eat) — Product Requirements Document

**버전**: 1.0  
**최종 수정**: 2026-04-09  
**모토**: "운동과 식단 — 운동은 먹는 것까지다"

---

## 1. 제품 개요

### 1.1 프로덕트 비전

W.E는 운동과 식단을 하나의 앱에서 종합 관리하는 AI 기반 피트니스 PWA다. 번핏(운동 기록)과 팻시크릿(식단 기록)의 핵심 기능을 결합하고, Claude AI를 통해 개인 맞춤 루틴·식단 추천을 제공한다.

### 1.2 타겟 사용자

- 헬스장을 다니며 운동과 식단을 함께 관리하고 싶은 20~40대
- 기존에 두 개 이상의 앱을 병행 사용하던 사람
- AI 기반 맞춤 루틴이 필요한 입문~중급 트레이니

### 1.3 핵심 차별점

| 기능 | 번핏 | 팻시크릿 | W.E |
|------|------|---------|-----|
| 운동 기록 | O | X | O |
| 식단 기록 | X | O | O |
| AI 루틴 생성 | X | X | O |
| AI 식품 사진 분석 | X | X | O |
| 탄수화물 사이클링 | X | X | O |
| 간헐적 단식 타이머 | X | X | O |
| PWA (앱 설치) | X | X | O |

## 2. 핵심 기능 요구사항

### 2.1 인증 & 온보딩

**목표**: 진입 장벽을 낮추고 AI 루틴 생성에 필요한 기본 정보를 수집한다.

**기능 목록**:
- Google OAuth 소셜 로그인 (이메일/비밀번호 미지원)
- 5단계 온보딩 플로우
  1. 목표 선택 (다이어트 / 근육 증가 / 체력 향상 / 체중 유지)
  2. 기본 신체 정보 (성별, 나이, 키, 몸무게)
  3. 활동량 (비활동적 / 가벼운 활동 / 보통 / 활동적 / 매우 활동적)
  4. 운동 환경 (기구 종류, 주 몇 회)
  5. 식이 제한 (비건, 유제품 제한 등)
- 온보딩 완료 즉시 AI 루틴 자동 생성

**제약**:
- 온보딩 미완료 유저는 서비스 접근 불가
- 이미 가입한 유저는 온보딩 스킵

---

### 2.2 운동 기록

**목표**: 헬스장에서 실시간으로 운동을 기록할 수 있는 빠르고 직관적인 UI를 제공한다.

**기능 목록**:

#### 운동 세션
- 세션 타이머 (시작 / 일시정지 / 재개 / 종료)
- 페이지 이탈 후 복귀 시 타이머 자동 복원 (localStorage)
- 종료 시 동기부여 팝업 (총 시간, 총 볼륨 표시)

#### 종목 관리
- 프리웨이트 / 머신 탭 분류
- 머신은 브랜드별 필터링 (라이프피트니스, 해머스트렝스 등)
- 근육 부위별 필터 (가슴, 등, 어깨, 하체, 팔, 복근)
- 종목 검색
- 직접 종목 추가 (미리 없는 종목)

#### 세트 기록
- 세트 타입: 일반 / 워밍업 / 드롭셋
- 중량 (kg) + 반복수 입력
- 세트 완료 체크 (Framer Motion 애니메이션)
- 휴식 타이머 자동 카운트다운
- 이전 운동의 세트 기록 자동 로드 (참고용)
- 1RM 자동 계산 (Stone 공식)

#### 기록 조회
- 운동 상세 페이지 (세션별 결과 검토)
- 홈 화면에서 오늘 운동 요약 카드

---

### 2.3 식단 기록

**목표**: 음식 기록의 마찰을 최소화하고 영양소를 정확하게 추적한다.

**기능 목록**:

#### 음식 추가
- AI 사진 분석: 카메라/갤러리 사진 → 음식 자동 인식 → 영양 정보 반환
- 텍스트 검색: 로컬 DB → 정부 식품영양성분 DB → AI 추정 순서로 조회
- 수동 입력: 이름, 용량, 칼로리, 단백질, 탄수화물, 지방 직접 입력

#### 식사 기록
- 4가지 식사 타입: 아침 / 점심 / 저녁 / 간식
- 날짜 네비게이션 (과거 기록 조회/수정)
- 식사 항목별 수정·삭제
- 식사 템플릿 저장 및 빠른 재사용

#### 영양소 추적
- 일일 칼로리 게이지 (목표 대비 현재)
- 탄수화물 / 단백질 / 지방 그래프
- 탄수화물 사이클링: 운동일 ×1.3, 휴식일 ×0.7 자동 적용
- AI 기반 당일 식단 추천 팁

---

### 2.4 AI 기능

**목표**: 사용자 데이터 기반 개인 맞춤 인사이트와 루틴을 제공한다.

**기능 목록**:

#### 맞춤 루틴 생성
- 입력: 목표, 체격, 활동량, 보유 기구, 주 운동 횟수
- 출력: 주간 일정 (요일별 운동 포커스 + 세부 종목), 영양 목표치, 휴식일
- AI 모델: Claude Sonnet 우선, 크레딧 부족 시 Gemini 2.5 Flash 자동 폴백
- 루틴 재생성 가능 (설정 페이지)

#### 음식 사진 분석
- 입력: 이미지 파일 (jpg/png)
- 출력: 식품명, 예상 용량(g), 칼로리, 탄단지
- 인식 불가 시 수동 입력 폴백

#### 식단 추천
- 당일 운동량·영양소 섭취량 기반 AI 코멘트 생성

---

### 2.5 대시보드 & 통계

**목표**: 장기적인 변화를 시각화해 꾸준한 동기부여를 제공한다.

**기능 목록**:
- BMR / TDEE 계산 카드 (Mifflin-St Jeor 공식)
- 84일 운동 히트맵 (GitHub 잔디 스타일)
- 최근 7일 칼로리 바 차트
- 4주 칼로리 + 운동 횟수 복합 차트 (Recharts)
- 체중 변화 라인 차트 (최근 30개 기록)
- 통계 요약: 월간 운동 횟수, 주간 운동일, 오늘 칼로리, 현재 체중

---

### 2.6 신체 데이터

**목표**: 체성분 변화를 기록하고 눈바디와 함께 모니터링한다.

**기능 목록**:
- 체중 / 체지방률 / 근육량 수동 입력
- 날짜별 기록 저장
- 눈바디 사진 갤러리 (Supabase Storage)
- 체중 추이 라인 차트

---

### 2.7 간헐적 단식

**목표**: 단식 스케줄을 타이머로 쉽게 추적한다.

**기능 목록**:
- 단식 프로토콜 선택: 16:8 / 18:6 / 20:4 / OMAD(23:1)
- 단식 시작/종료 타이머
- 현재 상태 표시 (단식 중 / 식사 가능)

---

### 2.8 건의사항

**목표**: 사용자 피드백을 수집하고 관리자가 응답할 수 있는 채널을 제공한다.

**기능 목록**:
- 사용자: 건의사항 제출 (제목 + 내용)
- 사용자: 내 건의사항 목록 + 관리자 답변 확인
- 관리자: 전체 건의사항 조회 + 상태 변경 + 답변 작성

## 3. 비기능 요구사항

### 3.1 성능
- 첫 페이지 로드: 3초 이내 (LCP 기준)
- AI API 응답: 10초 이내 (타임아웃 처리 필요)
- 음식 검색: 500ms 이내 (로컬 DB 기준)

### 3.2 모바일 UX
- 최대 너비 430px, 중앙 정렬
- 하단 네비게이션 5탭 (홈 / 통계 / 식단 / 신체 / 마이)
- PWA: 홈 화면 추가, 오프라인 지원
- 다크 테마 고정

### 3.3 보안
- Supabase RLS(Row Level Security) 적용 — 모든 테이블
- 서버 컴포넌트에서만 Service Role Key 사용
- AI API Key는 서버 사이드에서만 노출

### 3.4 가용성
- AI 기능: Claude 불가 시 Gemini 자동 폴백
- 식품 검색: 정부 API 불가 시 로컬 DB 단독 동작 (Graceful Degradation)

## 5. 데이터 모델 개요

### 핵심 엔티티

```
user_profiles (사용자 프로필)
  └── workouts (운동 세션)
        └── workout_exercises (세션 내 종목)
              └── sets (세트 기록)

  └── meals (식사 기록)
        └── meal_items (음식 항목)

  └── body_stats (체성분 기록)
  └── ai_routines (AI 루틴)
  └── meal_templates (식사 템플릿)
        └── meal_template_items

exercises (운동 종목 마스터 DB)
foods (식품 마스터 DB)
suggestions (건의사항)
```

## 6. 출시 범위

### MVP (현재 구현 완료)
- Google 로그인 + 온보딩
- 운동 기록 (세션, 세트, 타이머)
- 식단 기록 (AI 사진 분석, 검색, 수동 입력)
- AI 루틴 생성
- 대시보드 (4종 차트)
- 신체 데이터 + 갤러리
- 간헐적 단식 타이머
- 탄수화물 사이클링
- 건의사항

### 향후 고려
- 소셜 기능 (친구 운동 기록 공유)
- Apple Health / Google Fit 연동
- 유료 플랜 (AI 기능 횟수 제한 해제)
- 영양제 관리
- 운동 영상 가이드
