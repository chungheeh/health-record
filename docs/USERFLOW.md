# W.E — User Flow Document

**버전**: 1.0  
**최종 수정**: 2026-04-09

---

## 1. 신규 사용자 플로우

```
앱 접속
  └── /login
        └── Google 로그인 버튼
              └── Google OAuth
                    └── /auth/callback
                          └── user_profiles 없음
                                └── /onboarding
                                      ├── Step 1: 목표 선택
                                      │     (다이어트 / 근육 증가 / 체력 향상 / 유지)
                                      ├── Step 2: 기본 정보
                                      │     (성별, 나이, 키, 몸무게)
                                      ├── Step 3: 활동량
                                      ├── Step 4: 운동 환경
                                      │     (기구 종류, 주 운동 횟수)
                                      └── Step 5: 식이 제한
                                                  └── AI 루틴 자동 생성
                                                        └── / (홈)
```

---

## 2. 기존 사용자 플로우

```
앱 접속 (로그인 상태)
  └── / (홈)
        ├── 오늘 운동 요약 카드
        ├── 오늘 칼로리 요약
        └── AI 루틴 미리보기
```

---

## 3. 운동 기록 플로우

```
홈 → "운동 시작" 버튼
  └── /workout/new
        ├── 타이머 시작 (자동)
        │
        ├── 종목 추가
        │     ├── 탭 선택: 프리웨이트 / 머신
        │     ├── 근육 부위 필터
        │     ├── (머신) 브랜드 필터
        │     ├── 검색
        │     └── 직접 추가
        │
        ├── 세트 기록 (종목당 반복)
        │     ├── 세트 타입 선택 (일반/워밍업/드롭셋)
        │     ├── 중량 + 반복수 입력
        │     ├── 세트 완료 체크 → 휴식 타이머 자동 시작
        │     └── 다음 세트 추가
        │
        └── 운동 종료
              └── 완료 팝업 (총 시간, 볼륨)
                    └── /workout/[id] (결과 상세)
```

---

## 4. 식단 기록 플로우

```
하단 탭 → 식단
  └── /diet
        ├── 날짜 네비게이션 (← 오늘 →)
        ├── 영양소 게이지 확인
        │
        └── 식사 타입 선택 (아침/점심/저녁/간식)
              └── "+ 추가" 버튼
                    └── /diet/add
                          ├── 사진 촬영/업로드
                          │     └── AI 자동 분석
                          │           └── 결과 확인 → 저장
                          │
                          ├── 텍스트 검색
                          │     └── 검색 결과 선택 → 용량 조정 → 저장
                          │
                          └── 수동 입력
                                └── 이름, 용량, 영양소 직접 입력 → 저장
```

---

## 5. AI 루틴 플로우

```
하단 탭 → (홈에서 루틴 카드)
  └── /routine
        ├── 현재 AI 루틴 표시
        │     ├── 영양 목표 (칼로리, 탄단지)
        │     └── 주간 일정 (요일별 포커스 + 세부 종목)
        │
        └── 루틴 재생성 → /settings
              └── 프로필 수정 후 재생성 버튼
                    └── AI 호출 → 새 루틴 저장 → /routine
```

---

## 6. 신체 데이터 플로우

```
하단 탭 → 신체
  └── /body
        ├── 체중/체지방/근육량 입력 → 저장
        ├── 체중 추이 차트 확인
        └── 눈바디 사진 업로드
              └── 갤러리 표시
```

---

## 7. 간헐적 단식 플로우

```
홈 → 단식 카드 (또는 직접 /fasting 접근)
  └── /fasting
        ├── 프로토콜 선택
        │     (16:8 / 18:6 / 20:4 / OMAD)
        ├── 단식 시작
        │     └── 타이머 진행 중...
        │           ├── 단식 중 (남은 시간 표시)
        │           └── 식사 가능 (먹을 수 있는 시간 표시)
        └── 단식 종료
```

---

## 8. 건의사항 플로우

```
마이 → 건의사항
  └── /suggestions
        ├── 내 건의사항 목록 확인
        │     └── 관리자 답변 확인
        └── 새 건의사항 작성
              └── 제목 + 내용 → 제출

관리자:
  └── /admin
        ├── 전체 건의사항 조회
        ├── 상태 변경 (검토중 / 해결됨)
        └── 답변 작성
```

---

## 9. 설정 플로우

```
하단 탭 → 마이
  └── /my
        ├── 프로필 정보 확인
        ├── 총 운동 횟수 확인
        └── 설정 → /settings
              ├── 프로필 편집 (목표, 신체정보, 활동량 등)
              ├── BMR/TDEE 실시간 계산 확인
              └── 루틴 재생성 버튼
```

## 10. 에러 케이스

| 상황 | 처리 |
|------|------|
| 미인증 접근 | /login 리다이렉트 |
| AI API 실패 | Gemini 폴백 → 모두 실패 시 에러 메시지 |
| 식품 검색 결과 없음 | 수동 입력 유도 |
| 정부 API 키 만료 | 로컬 DB만 검색 (graceful degradation) |
| 사진 분석 실패 | 수동 입력 폴백 |
| 운동 종목 없음 | 종목 추가 안내 |
